# The Flow Game

The Flow Game is a browser-based puzzle game inspired by charity: water.

## Concept
Players rotate pipe tiles to connect a water source to a storage tank before time runs out.

## Features
- Interactive tile rotation
- Score tracking
- Timer-based gameplay
- Win and lose conditions
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript

## Future Improvements
- Multiple levels
- Animated water flow
- Sound effects
